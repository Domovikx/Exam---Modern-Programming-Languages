5 баллов - task 1

Написать программу решения квадратного уравнения. Корни только вещественные. Квадратное уравнение – это уравнение вида ax2 + bx + c = 0, где a не равно 0. Для решения квадратного уравнения можно использовать формулы:
$$x_+=\frac{-b+\sqrt{b^2-4ac}}{2a}$$
и
$$x_-=\frac{-b+\sqrt{b^2-4ac}}{2a}$$
где D = b2 - 4ac — дискриминант многочлена ax2 + bx + c. Если D > 0, то уравнение имеет два различных вещественных корня. Если D = 0, то корень один. Если D < 0, то нет решений.
